## Что это

`/case-team` — открытая мультиагентная система на базе Claude Code, которая выполняет работу
стратегической кейс-команды (PM + аналитик + финансист + маркетолог + дизайнер) за стоимость
подписки на нейросеть.

**На выходе — физические файлы:** `.xlsx` финмодель с формулами, `.pptx` презентация с
навигацией, `SOLUTION.md` с Decision Log, аналитика с источниками. Не текст в чате.

**Эта заявка собрана этой же командой `/case-team`** за день до дедлайна — мета-демо тезиса.

## Архитектура команды

```
                  ┌──────────────────┐
                  │  case-manager    │   ◄── куратор смысла
                  │  PM-оркестратор  │
                  └────┬─────────────┘
                       │ Agent tool (задача с контекстом)
                       ▼
        ┌──────────────┼──────────────┐
        ▼              ▼              ▼
   ┌─────────┐   ┌──────────┐   ┌──────────┐
   │analyst  │   │financier │   │marketer  │
   │research │   │финмодель │   │AJTBD     │
   └────┬────┘   └────┬─────┘   └─────┬────┘
        │ результат-.md в свою папку  │
        └──────────────┼──────────────┘
                       │ (cross-agent через blockers/, всегда через менеджера)
                       ▼
                ┌─────────────┐
                │case-designer│ ◄── конвертер формата
                │YAML.md→.pptx│
                └─────────────┘
```

**4 архитектурных решения:**

1. **Менеджер — куратор смысла внутри команды.** Решает, что класть на слайды; специалисты не пишут сразу в .pptx. Снимает 85% failure rate автономных агентов (Devin 15% completion на сложных задачах — независимые тесты).
2. **На выходе — физические рабочие файлы**, не текст в чате. `.xlsx` с формулами, `.pptx` готов к показу, `.md` Decision Log с источниками.
3. **Дотренировка на 2 реальных кейс-чемпионатах** — Cup Moscow 2024 (Axenix) + Inno Case Hack 2025 (Askona). Из них извлечено 26 методологических логик.
4. **Иерархия источников в каждом правиле агента:** эталон → бизнес-классика (Osterwalder, MoSCoW, AS-IS/TO-BE) → эвристика менеджера. Снижает галлюцинации.

## Главная работа клиента (AJTBD)

**Big Job:** принимать сильные обоснованные решения **и** проводить их через стейкхолдеров (инвесторы / совет / партнёры / команда), чтобы развивать бизнес.

**Что выполняет `/case-team` полностью:** подготовить полный набор материалов решения бизнес-кейса под защиту (презентация + финмодель + аналитика + позиционирование + Decision Log).

**Что остаётся за человеком:**
- Принять финальное go/no-go решение
- Защитить решение лично перед стейкхолдерами (живая коммуникация)
- Внедрить стратегию операционно
- Договориться с конкретными партнёрами
