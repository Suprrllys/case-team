### 1.4 Численность МСП в РФ (верификация TAM)

| Метрика | Значение, янв 2026 | Источник |
|---|---|---|
| Всего субъектов МСП | **6.835 млн** (+247 тыс YoY) | Ведомости 25.12.2025 [vedomosti.ru](https://www.vedomosti.ru/economics/articles/2025/12/25/1166484-chislo-subektov-msp-uvelichilos) |
| Из них ИП | ~4.4 млн | ФНС реестр МСП [rmsp.nalog.ru](https://rmsp.nalog.ru/) |
| Из них организаций (ООО, АО) | ~2.1 млн | ФНС реестр МСП |
| Микропредприятия (<15 чел) | ~6.46 млн (95.6%) | Корпорация МСП [corpmsp.ru](https://corpmsp.ru) |
| Малые (15-100 чел) | ~225 тыс (3.3%) | Минэкономразвития |
| Средние (100-250 чел) | ~21 тыс (0.3%) | Минэкономразвития |
| Крупные (>250 чел) | ~20-25 тыс (вне МСП-реестра) | TAdviser оценка |

**Расчёт TAM (формула):**

```
ЦА «компании со стратегическими задачами регулярно» =
   малые (225k) + средние (21k) + крупные (20k) - дубли
=  ~266 тыс компаний РФ
```

**Микропредприятия исключаем:** у них стратегические задачи редкие, обычно решаются фаундером самостоятельно, бюджет на консалтинг отсутствует. Это **жёсткая фильтрация**, верифицирующая консервативность оценки.

**Сверка с независимым источником:** RAEX-RR в рэнкинге 2025 показывает выручку 140 млрд ₽ при ~70 крупнейших клиентов на топ-12 компаний; средний чек = 140 млрд / N клиентов. При среднем чеке ~2-5 млн ₽ это даёт **30-70 тыс активных корпоративных клиентов консалтинга в РФ** — то есть из 266k наших целевых компаний только ~10-25% сейчас покупают консалтинг. **Остальные 75-90% — недополученный спрос из-за барьера стоимости.** Это валидирует TAM-as-Outcome из unit-economics.

**Sensitivity TAM** (если фильтр был слишком агрессивный):
- Консервативный (только малые+средние, без крупных): **246k** → TAM в деньгах 17-75 млрд ₽
- Базовый: **266k** → 18-82 млрд ₽
- Оптимистичный (+ микро 15-50 чел из сегмента «активный микробизнес»): **~330k** → 23-100 млрд ₽

**Вывод:** оценка `266k / 80 млрд ₽` устойчива на нижнем-среднем краю — это правильный консервативный якорь.

**Что это значит для нас:** ставка `1 100 ₽/час` для middle-команды РФ — это **консервативная** оценка. Если жюри спросит «а у нас на рынке дороже / дешевле?» — у нас есть источники по обе стороны диапазона. Якорь экономии 89-98% выдерживает любые разумные движения ставки.

### 2.A.1 Manus AI

- **Позиционирование** (с лендинга): «Autonomous general-purpose AI agent» — единый агент, который **выполняет задачи end-to-end** (research, аналитика, web-scraping). Главный месседж — «agent that gets things done».
- **ЦА:** solo founders, исследователи, мелкие команды; **не корпоративный сегмент** (security, governance отсутствуют у Free/Pro).
- **Главная Job:** «Когда мне нужно собрать данные / провести исследование / автоматизировать рутину, хочу запустить агент и забыть, чтобы получить результат без своего участия».
- **Фичи:** browser automation, web research, CSV/spreadsheet generation, file outputs.
- **Цены:** Free / **$19 Basic (100 credits)** / **$39 Plus (300 credits)** / **$199 Pro (19 900 credits)** / Team — [felloai.com](https://felloai.com/manus-ai-pricing/), [lindy.ai/blog/manus-ai-pricing](https://www.lindy.ai/blog/manus-ai-pricing)
- **Соцдоказательства:** ProductHunt, Trustpilot — **полярные**: «excellent researcher» vs «scam, deleted my credits». [trustpilot.com/review/manus-ai.sbs](https://www.trustpilot.com/review/manus-ai.sbs)

**Отзывы (критично для дифференциации):**

| Что хвалят | За что жалуются |
|---|---|
| Modulo единый агент, не надо думать про оркестрацию | **Hallucinated progress** — говорит «100% complete», когда реально 37% (Reddit) [riotimesonline.com](https://www.riotimesonline.com/manus-a-i-review-14-failures-in-two-weeks-of-testing/) |

| | **$2 380 в credits молча удалили при downgrade плана** — описано в Reddit-треде [trickle.so](https://trickle.so/blog/manus-ai-review) |

**Вывод:** Manus — это general-purpose агент с серьёзной reliability-проблемой на сложных задачах. **Не выдаёт `.xlsx`/`.pptx` пакетом с источниками**. Не имеет специализированных ролей.

- **Фичи:** 400+ интеграций (Gmail, Slack, HubSpot, Salesforce, Notion), голосовые агенты, multi-step workflows.
- **Цены:** Free (400 credits/мес) / **$19.99 Starter (2 000 cred)** / **$49.99 Pro (5 000 cred)** / Business custom / Enterprise. Доп. кредиты $10/1000. **Скрытая onboarding fee ~$1 500** — не указана на pricing-странице. ### 2.A.3 CrewAI

- **Позиционирование:** «The leading multi-agent platform» — open-source framework + платное Enterprise (AMP). Для **разработчиков**, не для конечных клиентов.
- **ЦА:** Python-разработчики, инженеры AI-команд в enterprise; **40% Fortune 500** компаний используют (по их собственным заявлениям).
- **Главная Job:** «Когда мне нужно построить multi-agent систему под свой бизнес-процесс, хочу удобный фреймворк с ролями + Flows».
- **Фичи:** Crews (role-based agents) + Flows (event-driven workflows); CrewAI Studio (low-code); AMP (observability, control plane).
- **Цены:** Open-source = бесплатно; **$25/мес Pro (100 executions)**; Enterprise **$60-120 тыс/год**. [crewai.com/pricing](https://crewai.com/pricing), [zenml.io/blog/crewai-pricing](https://www.zenml.io/blog/crewai-pricing)
- **Соцдоказательства:** GitHub stars (open-source); **отсутствие отзывов на G2, Trustpilot, Capterra** — несмотря на «40% Fortune 500» — Lindy и независимые обзоры это критикуют.

**Отзывы (Reddit + Medium):**

| Что хвалят | За что жалуются |
|---|---|
| Быстро прототипировать | **Debugging difficult** — нельзя писать unit-tests |
| Открытый код, видна логика | **Latency** issues при real-time нагрузке |

### 2.C.1 ChatGPT / Claude.ai (универсальные AI-чаты)

- **Позиционирование:** general-purpose AI assistant. **Самый частый «дефолтный» инструмент** фаундера: «спрошу у ChatGPT/Claude».
- **ЦА:** массовый — индивиды, разработчики, аналитики, маркетологи.
- **Главная Job:** «Когда у меня вопрос / задача, хочу ответ в чате».
- **Фичи:** беседа, генерация текста, мультимодальность, плагины/MCP, code interpreter; **Claude Artifacts** + **Claude for PowerPoint connector** (можно генерить HTML/.pptx прямо в чате).
- **Цены:** ChatGPT Free / Plus $20/мес / Team / Enterprise; Claude.ai Free / Pro $20/мес / Max / Team.
- **Соцдоказательства:** **800M+ пользователей** (ChatGPT) — массовый дефолт. Это и есть наш **главный косвенный конкурент** в голове клиента-фаундера.

**Что Claude/ChatGPT **могут**:**
- Генерить текст слайдов, идеи структуры
- Создать `.pptx` через Claude Design (HTML→PPTX) — есть PowerPoint connector с осени 2025 [claude.com/claude-for-powerpoint](https://claude.com/claude-for-powerpoint)
- Дать список конкурентов через web search

**Что Claude/ChatGPT **не могут** (это и есть зазор):**
- Не делают **методологически структурированный пакет** (xlsx + pptx + md по единой логике)
- **Нет разделения ролей** — один универсальный assistant
- **Не дотренированы на эталонах кейс-чемпионатов** — генерация шаблонна
- Не верифицируют цифры через **иерархию источников** (эталон → классика → эвристика)
- Каждый запуск **даёт разный результат** — нет воспроизводимости
- **Нет финмодели с формулами и sensitivity** — только текстовая «оценка»

**Ключевая ссылка по ограничениям:** [mindstudio.ai/blog/gamma-vs-chatgpt-vs-claude-vs-google-slides](https://www.mindstudio.ai/blog/gamma-vs-chatgpt-vs-claude-vs-google-slides-ai-presentation-tool-comparison), [prezent.ai/blog/claude-for-powerpoint](https://www.prezent.ai/blog/claude-for-powerpoint) — Claude генерит **HTML, не .pptx без конвертации**; ChatGPT — text-first, требует ручной переноса в слайды.

**Вывод:** ChatGPT/Claude — наш **главный когнитивный конкурент** («зачем мне `/case-team`, если можно спросить у ChatGPT?»). Дифференциация — через **методологию + физические артефакты + специализированные роли + дотренировку на эталонах**, не через «у нас AI лучше».

### 3.2 Главные выводы из матрицы
